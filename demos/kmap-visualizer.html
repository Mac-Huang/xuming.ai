How to Use

1. Select the number of variables (2-6)

2. Enter your Boolean function using truth table, minterms, or expression

3. Click "Generate K-Map" to visualize

4. Groups of 1s will be automatically highlighted

5. For 3+ variables, switch to 3D view for cube visualization
