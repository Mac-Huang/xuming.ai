Project Updates: Version 4.0

Iterating Based on Peer and TA Feedback

Summary of Modifications

Following peer feedback from classmates in mid-April 2026, we have implemented several key enhancements to improve the transparency, interactivity, and readability of the ML Project page. Below are the updates in Version 4.0:

1. Interactive UI Enhancements (Feedback from Evelyn and Wuziyang)

  • Toggle Camera Logic: The webcam button now toggles between "Start Camera" and "Stop Camera," allowing users to easily deactivate the model when finished.
  • Transparency Disclaimer: Added a prominent "Transparency Note" before the webcam demo to clarify the model's limitations (trained only on two identities) and its privacy-first local processing.

2. Readability & Organization (Feedback from Emilia and Kavyasri)

  • Structural Subheadings: The project statement has been broken down into six distinct sections (e.g., "Inspiration: Unmasking AI", "Privacy through Edge Computing") to eliminate dense text blocks and improve the reading experience.

3. Previous Updates (Version 3.0)

  • Visual Overhaul: Introduced a modern indigo color palette and consistent navigation across all pages.
  • Tech Hero Expansion: Added Michael Running Wolf as Anisha's Tech Hero alongside Joy Buolamwini.
